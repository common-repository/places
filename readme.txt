Interactive maps in a block with multiple markers and their popups. Manage locations in the Places post type.

== Description ==

*   No API key needed, no limitation
*   No GPS coordinates needed, just type any address 
*   Create as many maps as you need
*   Beautiful, interactive, responsive, and lightweight maps
*   Display any map in your posts and pages using the block Places
*   Create as many locations as you need in the dedicated post type and add content
*   Centers the maps in the middle of all displayed markers by default
*   On click on a location, recenter the map on the selection
*   The data can be loaded asynchronously in front-end

== Installation ==

1. Upload `places` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add locations in the Places post type, entering for each their name and address
1. Add the block Places in your posts and pages to select which location to display
